## Tips for Better Results

### Voice Recording Tips

**Clarity:**
- Speak slowly and clearly
- Avoid background noise
- Use simple words when possible
- Pause between different elements

**Structure:**
- Start with the main subject
- Add descriptive details
- Mention the style or mood
- Keep it under 30 seconds

### Prompt Engineering

**Effective Patterns:**

1. **Subject + Setting + Style**
   - "A lighthouse on a rocky coast, stormy weather, dramatic lighting"

2. **Subject + Action + Environment**
   - "A surfer riding a wave at sunset on a tropical beach"

3. **Object + Characteristics + Context**
   - "A vintage typewriter with aged paper, sitting on an old wooden desk"

**Quality Enhancers:**
Add these words to improve image quality:
- "high quality"
- "detailed"
- "professional"
- "8k resolution"
- "photorealistic"

### What to Avoid

**Problematic Prompts:**
- Very long descriptions (>50 words)
- Conflicting instructions
- Inappropriate content
- Too many subjects in one image
- Overly abstract concepts

**Technical Issues:**
- Mumbling or unclear speech
- Very quiet or loud recording
- Background music or noise
- Multiple languages in one prompt

## Testing Your Prompts

### Quick Test Process

1. **Start Simple**: Use basic prompts first
2. **Add Details**: Gradually add more descriptive elements
3. **Note Results**: Keep track of what works well
4. **Iterate**: Refine based on generated images

### Evaluation Criteria

**Good Results:**
- Image matches the description
- High visual quality
- Appropriate composition
- Clear subject matter

**Areas for Improvement:**
- Unclear or distorted features
- Wrong colors or lighting
- Missing elements from description
- Low image quality

## Troubleshooting Prompts

If your prompts aren't generating good images:

1. **Simplify**: Remove complex elements
2. **Clarify**: Use more specific descriptions
3. **Restructure**: Change the order of elements
4. **Reference**: Look at successful examples above

Remember: The AI Agent in the workflow will help optimize your prompts, but starting with clear, descriptive speech gives the best foundation for great images!
